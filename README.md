### Installing
  **Arch:** [fet.sh-git](https://aur.archlinux.org/packages/fet.sh-git/) (AUR)  
  **Other Distros:** copy `fet.sh` to somewhere in $PATH

### Customization
`fet.sh` can be configured using environment variables, for example:
```sh
$ separator=" > " fet.sh
```
Supported options are:
- `accent`
- `separator`
